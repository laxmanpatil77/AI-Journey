# Day 2: If-Else Operators in Python

This Python script demonstrates the use of if-else conditional statements and operators to perform various checks on user-input numbers.

## Features

The script includes four main examples:

### 1. Even or Odd Check
- Prompts the user to enter a number.
- Uses the modulo operator (`%`) to determine if the number is even or odd.
- Prints the result accordingly.

### 2. Positive, Negative, or Zero Check
- Asks for a number input.
- Uses if-elif-else to classify the number as positive, negative, or zero.
- Handles the edge case of zero separately.

### 3. Largest of Three Numbers
- Takes three numbers as input.
- Compares them using logical operators (`and`) to find the largest.
- Outputs the largest number.

### 4. Divisibility by 5 and 3
- Requests a number.
- Checks if it's divisible by both 5 and 3 using modulo and logical `and`.
- Prints whether it meets the criteria.

## How to Run

1. Ensure you have Python installed.
2. Run the script: `python ifelse-operators.py`
3. Follow the prompts to enter numbers as requested.

## Example Output

```
Enter a number: 4
The number 4 is Even.

Enter a number: -5
The number -5 is Negative.

Enter a 1st number: 10
Enter a 2nd number: 20
Enter a 3rd number: 15
The number 20 is largest number.

Enter a number: 15
A number is divisible by both.
```
