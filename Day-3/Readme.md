# Day 3: Loops in Python

In this day, I learned how to use loops in Python and how they work.

I practiced both `for` loop and `while` loop and understood when to use each one.

## What I Learned

- for loop → used when number of iterations is known
- while loop → used when condition-based looping is required
- use of `range()` function
- how to use `%` and `//` for digit-based problems

## Problems I Solved

- Print numbers from 1 to 10
- Print even numbers
- Sum of N numbers
- Factorial of a number
- Reverse a number
- Check palindrome number
- Count digits and sum of digits

## Key Understanding

I learned that:
- Some problems (like reverse number) are better solved using `while` loop
- Using `for` loop in such cases makes code complex

## Goal

To build strong logic and understand loop behavior deeply for problem solving and future DSA.
