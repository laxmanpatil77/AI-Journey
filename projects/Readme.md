# Project 1: Smart Student Result System

This is my first Python mini project where I used basic programming concepts to build a student result system.

## Features

- Takes student name and marks of 3 subjects
- Calculates total and average marks
- Assigns grade based on average
- Checks pass/fail condition based on each subject

## Concepts Used

- Variables
- Input/output
- Arithmetic operations
- if-else conditions
- Logical operators (and)

## Important Logic

A student will fail if any subject marks are less than 35.

## What I Learned

- How to apply conditions in real problems
- How to handle multiple inputs
- Difference between average-based logic and subject-wise logic
- Importance of handling edge cases

## Goal

To move from basic coding to real problem-solving using Python.
